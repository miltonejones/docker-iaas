## Summary

**Issue:** Remove "Total containers" and "Resolved issues" cards from the home page.

**Root cause:** The home page (`/app/web/src/pages/Home.tsx`) rendered 7 stat cards in a grid, including "Total containers" (showing all containers regardless of state) and "Resolved issues" (showing resolved assistant-created issues). The issue requested removing both.

**Changes made:**

1. **`/app/web/src/pages/Home.tsx`:**
   - Removed the `totalCount` prop from the `Props` interface and function signature (no longer needed)
   - Removed the `resolvedIssueCount` state variable and its setter (was only used by the removed card)
   - Removed the `setResolvedIssueCount(counts.resolved)` call from `loadCounts()` (still fetches issue counts for the "Open issues" card)
   - Removed the "Total containers" card JSX (`<button>` linking to `/containers`, displaying `totalCount`)
   - Removed the "Resolved issues" card JSX (`<div>` displaying `resolvedIssueCount`)

2. **`/app/web/src/App.tsx`:**
   - Removed the `totalCount={containers.length}` prop from the `<HomePage>` component usage

**Remaining home page cards (5):** Running containers, Saved functions, Buckets, Gateway routes, Open issues.
