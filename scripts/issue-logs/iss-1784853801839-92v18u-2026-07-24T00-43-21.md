## Response

## Summary

**Root cause:** The delete button in the saved-sessions offcanvas panel (`App.tsx:501`) uses classes `btn btn--ghost btn--sm offcanvas-session-row__delete`. The `.btn` base class sets `border: 1px solid var(--border-strong)`, and `.btn--ghost` only softens it to `border-color: var(--border)` — neither removes the border entirely. The `.offcanvas-session-row__delete` CSS rule only declared `flex-shrink: 0`, so nothing was overriding the inherited border.

**Fix:** Added `border: none` to `.offcanvas-session-row__delete` in `web/src/styles.css` (line 2901), which overrides the border inherited from the `.btn` / `.btn--ghost` classes.
